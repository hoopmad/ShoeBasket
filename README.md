## :iphone: Project Overview
### 화면 구성
1. 농구화를 브랜드별로 정리한 메인 페이지
2. 농구화에 대한 기본적인 정보를 알 수 있는 페이지
3. 전체 농구화를 볼 수 있는 페이지
